The **Bike Sale Dashboard** is a fully interactive Excel dashboard designed to analyze customer purchasing behavior and identify trends related to bike sales. It transforms raw sales data into meaningful visual insights using Pivot Tables, Pivot Charts, Slicers, KPIs, and Excel Dashboard Design techniques.
