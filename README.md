# 🌟 Project Overview

The **Bike Sale Dashboard** is a fully interactive Excel dashboard designed to analyze customer purchasing behavior and identify trends related to bike sales. It transforms raw sales data into meaningful visual insights using Pivot Tables, Pivot Charts, Slicers, KPIs, and Excel Dashboard Design techniques.

The dashboard enables users to filter data dynamically and gain quick insights into customer demographics, education, occupation, ownership patterns, regional distribution, and purchasing behavior.

# 📈 Key Business Insights

📌 Male customers have a slightly higher children count than female customers.

📌 Bachelor's degree holders represent the highest number of children.

📌 Customers without bike purchases slightly outnumber bike purchasers.

📌 Car ownership varies according to commute distance.

📌 Professional occupations contribute a significant share of car ownership.

📌 North America shows the highest regional age distribution.
